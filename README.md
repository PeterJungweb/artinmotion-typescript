# ArtInMotion

ArtInMotion is a web platform for selling unique artwork.  
The project is currently under active development and is designed to provide a smooth and modern purchasing experience for art.

The platform allows users to browse artwork, add items to a cart, and securely complete purchases online.

---

## Project Status

**Status:** In development  
**Last update:** March 3, 2025

Current focus of development:

- Orders system
- Orders page implementation
- Backend integration improvements
- Checkout flow stability

The project was initially started in **JavaScript** but was later **fully migrated to TypeScript** to improve maintainability, scalability, and type safety.

## Tech Stack

### Frontend
- TypeScript
- Modern JavaScript frameworks
- Responsive UI design
- Browser Developer Tools for debugging and performance monitoring

### Backend
- Supabase (database & backend services)
- REST-style API structure
- Controllers and services for structured backend logic

### Payments
- Stripe integration for secure checkout and payment processing

### Version Control
- Git for version control and structured development
